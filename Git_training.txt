-------------------- Git Commands --------------------

1) Get code from git repository:
- git clone $SSH_LINK


2) Create Branch:
- git checkout -b $NEW_BRACH_NAME

NOTE: 
Branch name must be $odoo_version-$Tracking_number-$Task_name-BAD-$Emp_short_tag
15.0-T009-Python-Training-BAD-SUM


3) Check status
-> After changing in existing file or after creating new files
- git status

Note:
-> Changed files or new files display in red color.


4) Check difference in the files
-> After chaning in existing file or after creating new files
- git diff


5) Remove all code changes
- git stash


6) Get changes back that we have stash
- git stash pop


5) Add file changes.
- git add --all or git add .
-> Multiple file changed and want to add only specific file:
- git add $File_Name


6) Git commit
->
TAGS = ADD: Add new files, IMP: Make changes in existing files, REM: Remove any files, i18n: Update PO(translation file.)
- git commit -am "[TAGS][Tracking-Number] $Module-Name: Proper Commit Message."


7) Git Push
-> Push your code to the git in your branch.
- git push


8) Check branch of repository.
-> git branch --all


9) When create a new branch and then doing pull for 1st time.
